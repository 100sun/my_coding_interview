### Topics
#### Big O

1 < log n (binary search) < n < n log n (merge sort) < n2(bubble sort) < 2n < n! <br/>
[Cheat Sheets](https://www.bigocheatsheet.com/)
<br/>-> You have to check time-complexity of the worst case ( 1ns = 10*(-9)s )


#### Data Structure
* Linear
    * hash / dictionary : O(1)
        * search
    * stack : O(n)
        FILO
    * queue : O(n)
        FIFO
    * linked list : O(n)

* Nonlinear
    * tree : O(1)
        * binary search tree
